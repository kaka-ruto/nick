# Library and Reader Experience

## Public discovery

- `/` introduces Chapterwan and featured books
- `/library` provides full browse/search

## Reader output

Published revisions appear in Chapterwan's web reader.
This is the canonical reading surface for readers.

## What readers should see

- clear cover and metadata
- stable navigation
- consistent typography
- fast load and readable structure

## What operators should verify

- correct title/subtitle/author/category/tags
- expected pricing and visibility
- expected cover behavior (uploaded or default themed cover)
