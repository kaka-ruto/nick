# Chapterwan CLI reference

Install and manage Chapterwan.

To run the `chapterwan` command in a terminal, connect to your server using either `SSH` or a web-based cloud console.

## Usage:

`chapterwan` - [command]

## Manage passwords

`password`  - manage passwords

## Manage automatic updates

`auto-update` - manage automatic updates

## Manage application data

`data` - manage application data

## Additional Commands:

`help` - Help about any command
`setup` - Re-run the initial setup of Chapterwan
`start` - Start Chapterwan
`status` - Show the current status
`stop` - Stop Chapterwan
`update` - Update Chapterwan to the latest version

## Flags:

`-h`, `--help`  - help for chapterwan

Use `chapterwan [command] --help` for more information about a command.
