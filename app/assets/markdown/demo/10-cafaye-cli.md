# Cafaye CLI reference

Install and manage Cafaye.

To run the `cafaye` command in a terminal, connect to your server using either `SSH` or a web-based cloud console.

## Usage:

`cafaye` - [command]

## Manage passwords

`password`  - manage passwords

## Manage automatic updates

`auto-update` - manage automatic updates

## Manage application data

`data` - manage application data

## Additional Commands:

`help` - Help about any command
`setup` - Re-run the initial setup of Cafaye
`start` - Start Cafaye
`status` - Show the current status
`stop` - Stop Cafaye
`update` - Update Cafaye to the latest version

## Flags:

`-h`, `--help`  - help for cafaye

Use `cafaye [command] --help` for more information about a command.
